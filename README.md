# Klantwebsite Template

Dit project is een basis-template voor simpele klantwebsites.

## Aanpassen per klant

- [ ] Bedrijfsnaam aanpassen
- [ ] Logo vervangen in de images-map
- [ ] Kleuren aanpassen in css/style.css bij :root
- [ ] Homepage teksten aanpassen
- [ ] Diensten aanpassen
- [ ] Over ons-pagina aanpassen
- [ ] Contactpagina aanpassen
- [ ] E-mailadres aanpassen
- [ ] Telefoonnummer aanpassen
- [ ] WhatsApp-link aanpassen
- [ ] Footergegevens aanpassen
- [ ] SEO titles aanpassen
- [ ] Meta descriptions aanpassen
- [ ] Alt-teksten van afbeeldingen aanpassen
- [ ] Domein koppelen in Vercel
- [ ] Mobiele versie controleren

## Bestandenstructuur

- index.html = homepage
- over-ons.html = over ons-pagina
- contact.html = contactpagina
- css/style.css = styling
- js/script.js = formuliermelding
- Images/ = logo en afbeeldingen
